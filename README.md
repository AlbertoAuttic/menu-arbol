
# Menú en Árbol

Creación de menú en árbol con React y react-treebeard.

## Ejecutar en desarrollo

Clona el repositório

```bash
  git clone https://github.com/AlbertoAuttic/menu-arbol.git
```

Go to the project directory

```bash
  cd my-project
```

Instalar dependencias

```bash
  npm install
```

Iniciar servidor

```bash
  npm run start
```
